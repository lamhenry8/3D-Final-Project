# Dungeon Leveling

## Game Concept

A third-person dungeon crawler: explore a castle dungeon, harvest resources from ore and monsters, gear up with what you find, and take on the dungeon boss.

## Objective

Collect basic resources from ores and monsters to gear up and fight the dungeon boss!

## Controls

| Action | Input |
| --- | --- |
| Move | WASD |
| Interact | E |
| Inventory | I |
| Mine | Left-click |
| Attack | Left-click |

## Gameplay Systems

- Inventory system
- Ore collecting
- Enemy drops

## Known Issues
- `BasicPickaxe` material was pointing at an HDRP shader in a URP-only project, causing textures to fail to render (fix: re-import the pack's URP material or reassign `Universal Render Pipeline/Lit`).
